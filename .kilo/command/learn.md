# Learning Agent Command

เรียก Learning Agent เพื่อศึกษาค้นคว้าความรู้ใหม่ๆ

## การใช้งาน
- `/learn {topic}` — ศึกษา topic ที่ระบุ
- `/learn --scan` — Scan แหล่งความรู้ทั้งหมด
- `/learn --retro {project-path}` — เรียนรู้จากโปรเจกต์ที่ทำเสร็จ
- `/learn --summary` — สรุปความรู้ที่สะสม
- `/learn --audit` — ตรวจสอบความเก่าของความรู้

ใช้ $ARGUMENTS ส่งพารามิเตอร์
